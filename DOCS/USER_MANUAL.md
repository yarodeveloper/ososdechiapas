# Manual de Usuario - Club Osos de Chiapas

Guía operativa para el personal del club y administradores.

## 1. Gestión de Prospectos (Leads)
Cada vez que alguien llene el formulario en la web principal:
1. Recibirás una notificación en tu panel.
2. Entra a **Prospectos**.
3. Usa el botón de **WhatsApp** para contactarlos.
4. Cambia su estado a "Contactado" para llevar el control.

## 2. Validación de Pagos
Cuando un padre de familia reporte un pago:
1. Ve a **Finanzas / Pagos**.
2. Los pagos por validar aparecerán resaltados en color naranja.
3. Revisa la imagen del comprobante.
4. Si es correcto, presiona **APROBAR**. El estatus cambiará automáticamente a "Pagado".

## 3. Captura de Estadísticas (Coaches)
1. En el Dashboard, verás partidos finalizados que requieren acción.
2. Presiona **Llenar Estadísticas**.
3. Ingresa los datos de yardas, tacleos y touchdowns.
4. No olvides marcar quién fue el **MVP** del partido.
5. Estos datos alimentarán automáticamente la **Playcard** que ven los padres.

## 4. Avisos y Notificaciones
Para enviar un mensaje a todos los padres:
1. Ve a **Avisos**.
2. Escribe el título y el mensaje (ej: "Entrenamiento suspendido por lluvia").
3. Al publicar, todos los usuarios lo verán en su portal familiar de inmediato.
